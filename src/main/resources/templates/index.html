TECH3S Web Application Skeleton is built on SPA (Single Page Application) model with the following technology stack:

  • Backend:

    1. Spring REST
    2. Spring Security
    3. JWT (JSON Web Token)
    4. Spring-Hibernate
    5. MySQL

  • Frontend:

    1. HTML (Hyper Text Markup Language)
    2. CSS (Cascading Style Sheet)
    3. Javascript
    4. jQuery
    5. Sammy

  • DevOps:

    1. Apache Server
    2. Amazon Web Services (EC2, S3)
    3. Jenkins
